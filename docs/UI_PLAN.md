# UI Plan

## 1. Home / Overview
Introduce CareGrid India, show backend readiness, dataset scale, and navigation into the main product areas.

## 2. Dashboard
Show KPI cards, trust distribution chart, readiness chart, and high-level national trust summary.

## 3. Facilities Explorer
Show a filterable facility table, map marker placeholder, trust/readiness badges, and facility detail drawer.

## 4. Impact / Trust Desert Analysis
Show priority states, state risk ranking, facility type reliability gaps, and national trust gap insights.

## 5. Search
Show ranked facility search results with matched fields, evidence summaries, and warning badges.

## 6. Agent Recommendations
Show a natural-language recommendation panel with interpreted intent, ranked recommendations, reasons, and safety note.
